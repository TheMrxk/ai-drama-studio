## 1. 项目概述

### 1.1 项目背景

**痛点**:
- 有故事想法，但不会写剧本
- 不懂剧本格式和结构
- 想创作短剧，但无从下手

**解决方案**:
一个 AI 驱动的短剧生成器，用户输入故事要素，AI 生成完整剧本。

### 1.2 项目目标

构建一个 Web 应用，实现：

1. ✅ **故事输入** - 用户输入故事设定、主角、剧情、结局
2. ✅ **AI 生成** - 调用大模型 API 生成剧本
3. ✅ **项目管理** - 每个剧本是一个项目，可追溯历史
4. ✅ **精美界面** - 现代化 UI，流畅体验

### 1.3 目标用户

| 用户类型 | 需求 |
|---------|------|
| **编剧爱好者** | 快速生成剧本初稿 |
| **短视频创作者** | 获取拍摄脚本 |
| **小说作者** | 改编小说为剧本 |
| **学生** | 学习剧本创作 |

### 1.4 使用场景

#### 场景 1: 创作一部都市爱情短剧

```
1. 用户打开网站
2. 点击"新建项目"
3. 输入：
   - 故事设定：现代都市，咖啡厅相遇
   - 主角：女主 - 咖啡师，男主 - 程序员
   - 剧情：每天买咖啡，逐渐产生感情
   - 结局：男主表白，两人在一起
4. 点击"生成剧本"
5. AI 生成完整剧本（5 场戏）
6. 用户可以修改、调整
7. 导出为 PDF/Word
```

#### 场景 2: 修改已有剧本

```
1. 打开项目列表
2. 点击某个剧本
3. 查看历史版本
4. 修改某一场戏
5. AI 重新生成该场次
6. 保存新版本
```

### 3.2 功能详细设计

#### 3.2.1 故事输入表单

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| **项目名称** | 文本 | ✅ | 剧本的名称 |
| **故事设定** | 长文本 | ✅ | 时代背景、地点、环境 |
| **主角特征** | 长文本 | ✅ | 姓名、性格、外貌、职业 |
| **剧情脉络** | 长文本 | ✅ | 主要情节发展 |
| **最终结局** | 长文本 | ✅ | 故事如何结束 |
| **剧集数量** | 数字 | ❌ | 默认 5 集 |
| **剧本风格** | 选择 | ❌ | 喜剧/悲剧/悬疑等 |

#### 3.2.2 剧本输出格式

```markdown
# 《项目名称》

## 基本信息
- 类型：都市爱情
- 集数：5 集
- 风格：温馨喜剧

---

## 第一集

### 第一场
**场景**: 咖啡厅 - 日 - 内
**人物**: 林小满（女主）、陈默（男主）

【动作】林小满正在吧台制作咖啡，陈默推门进来。

陈默
（走到吧台前）
一杯美式，谢谢。

林小满
（微笑）
好的，请稍等。

【动作】林小满熟练地制作咖啡，陈默在一旁看着。

陈默
你在这里工作多久了？

林小满
（抬头看他）
三年了。你是新面孔啊。

...

### 第二场
...
```

#### 3.2.3 版本管理

| 版本 | 时间 | 修改内容 | 操作 |
|------|------|---------|------|
| v1.0 | 2026-03-27 10:00 | 初始生成 | 查看/恢复 |

## 5. API 设计

### 5.1 API 概览

| 方法 | 端点 | 说明 |
|------|------|------|
| POST | /api/projects | 创建项目 |
| GET | /api/projects | 获取项目列表 |
| GET | /api/projects/:id | 获取项目详情 |
| PUT | /api/projects/:id | 更新项目 |
| DELETE | /api/projects/:id | 删除项目 |
| POST | /api/generate | 生成剧本 |
| GET | /api/projects/:id/versions | 获取版本历史 |
| POST | /api/export | 导出剧本 |

### 5.2 API 详细设计

#### 5.2.1 创建项目

```http
POST /api/projects

Request:
{
  "name": "咖啡厅的爱情",
  "setting": "现代都市，一家温馨的咖啡厅",
  "characters": "林小满（女主，25 岁，咖啡师，温柔善良）；陈默（男主，28 岁，程序员，内向害羞）",
  "plot": "陈默每天来买咖啡，逐渐对林小满产生好感。一次偶然的机会，两人开始聊天，发现彼此有很多共同爱好...",
  "ending": "陈默在咖啡厅向林小满表白，两人在一起",
  "episodes": 5,
  "style": "温馨喜剧"
}

Response:
{
  "id": "proj_123",
  "name": "咖啡厅的爱情",
  "created_at": "2026-03-27T10:00:00Z",
  "status": "draft"
}
```

#### 5.2.2 生成剧本

```http
POST /api/generate

Request:
{
  "project_id": "proj_123",
  "episode": 1,
  "prompt": "生成第一集剧本，包含 3 场戏"
}

Response:
{
  "task_id": "task_456",
  "status": "processing",
  "estimated_time": 30
}

# WebSocket 推送进度
{
  "task_id": "task_456",
  "status": "completed",
  "script": "# 第一集\n\n第一场\n场景：咖啡厅..."
}
```

#### 5.2.3 获取版本历史

```http
GET /api/projects/:id/versions

Response:
{
  "project_id": "proj_123",
  "versions": [
    {
      "version": "v1.0",
      "created_at": "2026-03-27T10:00:00Z",
      "changes": "初始生成",
      "content_hash": "abc123"
    },
    {
      "version": "v1.1",
      "created_at": "2026-03-27T10:30:00Z",
      "changes": "修改第一场对话",
      "content_hash": "def456"
    }
  ]
}
```
