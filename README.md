# AI Drama Studio - AI 短剧生成器

> 将你的想法变成精彩的 AI 短剧剧本

**版本**: v1.0.0  
**创建时间**: 2026-03-27  
**项目地址**: https://github.com/TheMrxk/ai-drama-studio

---

## 🎬 项目愿景

**每个人都是导演**。只需输入故事想法，AI 帮你生成完整的短剧剧本。

---

## 🌟 核心功能

### 1. 故事创作
- 输入故事设定、主角特征、剧情脉络、最终结局
- AI 智能编排成完整剧本

### 2. 项目管理
- 每个剧本是一个项目
- 记录所有修改和交互历史
- 版本对比和回溯

### 3. 精美界面
- 现代化 UI 设计
- 流畅的交互体验
- 实时预览

---

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/TheMrxk/ai-drama-studio.git

# 安装依赖
cd ai-drama-studio
npm install

# 启动开发服务器
npm run dev
```
